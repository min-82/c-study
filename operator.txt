1. 점(.) 연산자: "직접 접근"

    대상: 구조체 변수 (실제 데이터 덩어리)

    의미: "이 구조체 안에 있는 멤버를 가져와."

    상황: struct User user1; 처럼 일반적인 변수로 선언했을 때 사용합니다.

    예시: user1.name, userList[i].age (배열의 각 칸은 실제 데이터이므로)

2. 화살표(->) 연산자: "간접 접근"

    대상: 구조체 포인터 (데이터가 있는 주소값)

    의미: "이 주소가 가리키는 곳으로 가서, 그 안에 있는 멤버를 가져와."

    상황: malloc으로 할당받았거나 포인터 변수로 선언했을 때 사용합니다.

    예시: userPtr->name

3. 핵심 차이 비교
형태	문법	실제 의미	비유
변수	user.age	현재 위치의 데이터 확인	내 주머니 안의 물건 꺼내기
포인터	ptr->age	주소를 따라가서 데이터 확인	지도를 보고 찾아가서 문 열기
